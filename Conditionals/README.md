# Conditionals Guide

- IF

```python
if numero > 5:
	print('Is higher to 5')
```

- Else (si no)

```python
if numero > 5:
	print('Is higher to 5')
else
    pass ## Don't do nothing
```

- Else if (si no si)

```python
if numero > 5:
	print('Is higher to 5')
elif numero > 4:
	print('Is higher to 4')
else
    pass ## Don't do nothing

```

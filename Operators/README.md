## Operators guide

- Suma

```python
num1=1
num2=2
num3=num1+num2
```

- Resta

```python
num1=1
num2=2
num3=num1-num2
```

- División

```python
num1=1
num2=2
num3=num1/num2
```

- Residuo

```python
num1=4
num2=2
num3=num1%num2
```

- Multiplicación

```python
num1=4
num2=2
num3=num1*num2
```

- Potencia

```python
num1=4
num2=2
num3=num1**num2#16
```

- Trigonometria

```python
import math

math.sqrt(4) #2

math.pi #3.1415926535897931

math.sin(math.pi) #0

math.log(20) #2.995732273553991
```
